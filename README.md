# TALK APP SERVER

## Requirements

 * Python 3.4+
 * Django 2.1.1
 * PostgreSQL 9.3+

## Run Server

 1. Install and run Redis server
 2. Install and run PosgreSQL server

 4. Run Django server

    ```bash
    $ ./manage.py runserver
    ```

    Now the server is running on: http://localhost:8000
